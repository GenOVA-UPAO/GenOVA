# Docs-specs (DO)

Specs de entregables de documentación del product backlog (tipo `DO-N`).
Generadas por `spec_author` a partir de `sdd/backlog.md` (Fase B del import del backlog).

No confundir con `docs/` (documentación funcional/técnica que genera `doc_author`).

Ver índice y metadata en [`../backlog.md`](../backlog.md).
